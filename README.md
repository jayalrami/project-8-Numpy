# NumPy Analyzer

A **menu-driven Python application** that demonstrates core **NumPy** operations such as array creation, indexing & slicing, mathematical operations, combining & splitting arrays, searching, sorting, filtering, and statistical analysis.

This project is ideal for **students and beginners** learning NumPy through hands-on practice.

* Sort array elements

### 6. Aggregates, Statistics & Correlation

* Median
* Standard Deviation
* Variance
* Minimum & Maximum
* Percentile
* Correlation Coefficient (`np.corrcoef`)
